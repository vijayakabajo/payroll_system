Instructions
How to prepare your payroll file
  • 1 Download the Excel template using the button above. It contains all expected column headers.
  • 2 Fill in the employee_code column (required). This must match the employee code in the system.
  • 3 Earning overrides: basic, hra, ca, cca, bonus, medical, mobile, washing, special. Leave blank to use defaults from the salary configuration.
  • 4 Deduction overrides: pf, esi, pt, tds, loan, adv_sal. Leave blank for auto-calculated values.
  • 5 Attendance: paid_days and total_days (defaults to 30 if left blank).
  • 6 Leave data: cl, sl, lwp, ml, pl (defaults to 0 if left blank).
  • 7 Select the month and year for the payslips, then click Upload & Process.
  • 8 After processing, you'll see a progress report with per-row status. Download the error report for any failures.
Important

Only .xlsx files are supported. The first row must contain headers. Numeric columns must contain valid numbers. Existing payslips for the same employee/month/year may be overwritten.
